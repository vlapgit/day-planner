# day-planner - BtWAt
A minimalist, dark-themed daily planner BtWAt (Boring tasks, With A twist) that forces accountability instead of letting tasks quietly slide. Built around time blocks, not to-do lists — every task has a start time, an end time, and a real consequence if you miss it.

Features


Time-blocked scheduling — Tasks are scheduled in explicit time ranges (e.g. 1:00 am – 2:00 am), not just a flat list. This makes your day a timeline, not a backlog.
Live task states — Each task is visually color-coded by status:

✅ Completed — checked off, struck through, dimmed

🔴 Missed — red border and red timestamp if the time window passed with no completion

🔵 Running now — blue highlight with a live countdown (e.g. 33m left) for whatever's active in real time

⚪ Upcoming — future tasks shown in neutral gray



Weekly score — A blunt completion percentage at the bottom of the screen (e.g. 25%), paired with a plain-language verdict (COMPLETE FAILURE) instead of a softened stat. No participation trophies — the app tells you straight when a week didn't work.
Horizontal date strip — Quick day-to-day navigation across the month without leaving the task view.
Inline task management — Add, remove (×), and check off tasks directly from the day view; no separate edit screens.


Why this app is different than others on playstore

The main goal was to make a minimalist yet fully functional personalized day planner app.Most planner apps optimize for looking productive — streaks, soft nudges, "you're doing great!" messaging. This one is designed around honest failure states. A missed block stays visibly red until you deal with it, and the weekly score doesn't euphemize a bad week. The tradeoff is that it's blunt by design — if you want gentle accountability, this isn't that app, and that's the point.
